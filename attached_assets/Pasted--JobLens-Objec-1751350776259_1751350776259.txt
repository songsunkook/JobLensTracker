# 📘 **JobLens (가칭)** — 취업 데이터 통합 인사이트 서비스 기획서



---



## 🧭 목적 (Objective)



> 단순한 취업 공고 나열이 아닌,  

> **데이터 기반으로 ‘취업 시장의 흐름’을 보여주고**,  

> **취준생이 스스로 준비 방향을 잡을 수 있도록 돕는 서비스**  



---



## 🎯 주요 페르소나 (Personas)



### 👩‍💻 Persona 1 — 김지원 (24세, 대학생)

- **배경**: 수도권 4년제 컴공 재학, 취준 시작 단계

- **고민**: “어떤 역량을 준비해야 IT 서비스 기업에 취직할 수 있을까?”

- **Needs**: 산업별로 많이 요구되는 조건, 연봉 수준, 위치 등을 한눈에 보고 싶음



### 🧑‍🔬 Persona 2 — 박민호 (28세, 직장인)

- **배경**: 중소 SI에서 2년 차 경력, 이직 고려 중

- **고민**: “연봉 높고 워라밸 좋은 기업 어디 있을까?”

- **Needs**: 산업군/연봉 필터링 + 기업 문화 태그 + 위치 기반 시각화



### 🧑 Persona 3 — 이소영 (26세, 비전공 부트캠프 수료생)

- **배경**: 디자인 → 프론트엔드 전향, 비전공자

- **고민**: “비전공자에게 우호적인 기업은 어디일까?”

- **Needs**: 자주 등장하는 우대사항 키워드, 기업별 우대 문화 등 확인



---



## 🧩 핵심 가치 (Value Proposition)



- 🔍 **데이터 기반 통찰**: 수많은 공고를 분석하여 직무별 필수 요건, 우대사항 등 통계 제공  

- 🧭 **방향 제시형 탐색 경험**: 필터링을 통해 *자신이 관심 있는 분야에서는 어떤 역량이 중요한지* 파악  

- 🗺 **시각적 정보 전달**: 기업의 위치, 연봉 수준 등을 지도와 차트로 직관적으로 확인 가능  



---



## 🧱 서비스 구성 개요



| 영역 | 설명 |

|------|------|

| **1. 데이터 수집** | 다양한 취업 공고/연봉 정보를 주기적으로 수집 및 정제 |

| **2. 정보 탐색** | 조건에 맞는 기업을 필터링하고 요구 요건/연봉/위치 등을 통계로 확인 |

| **3. 시각화 기능** | 기업 분포 지도, 연봉 그래프, 키워드 클라우드 등 다양한 형태로 정보 제공 |



---



## 🧾 기능 명세 (Feature Specification)



### ✅ 1. 공고/연봉 데이터 수집



| 기능 | 설명 |

|------|------|

| 주요 채용 플랫폼 공고 수집 | 잡코리아, 원티드, 로켓펀치 등에서 채용 정보 정기 수집 |

| 연봉 정보 수집 | 잡플래닛, 크레딧잡 등에서 연봉 정보 확보 |

| 데이터 정제 및 구조화 | 기업명, 직무명, 기본요건, 우대사항, 지역, 연봉 등의 핵심 항목 추출 |

| 산업군/직무 분류 | 각 기업 또는 공고를 서비스/핀테크/제조 등으로 분류 (직무 중심 분류 포함) |

| 위치 정보 확보 | 기업 주소 → 지도 상 마커 표시를 위한 위치 정보 확보 |



---



### ✅ 2. 사용자 중심 정보 탐색 기능



| 기능 | 설명 |

|------|------|

| 필터 검색 | 사용자는 아래 조건들로 원하는 기업/공고 집합을 설정할 수 있음:<br>• 산업군 (ex. 핀테크, AI)<br>• 연봉 범위 (ex. 3천~5천)<br>• 지역 (ex. 서울, 원격)<br>• 경력 구분 (신입/경력)<br>• 근무형태 (정규직, 계약직 등) |

| 요구 요건 통계 제공 | 필터 조건에 해당하는 공고들에 대해:<br>• **가장 많이 등장한 기본요건**<br>• **자주 요구되는 우대사항**을 키워드 통계로 제공 |

| 연봉 분포 확인 | 조건에 해당하는 기업들의 **연봉 평균/범위/중간값** 등을 시각화 |

| 기업 즐겨찾기 | 사용자가 관심 있는 기업/공고를 저장해둘 수 있음 |

| 공고 상세 보기 | 공고별 상세 내용 확인 가능 (요약 + 원문 링크 제공) |



---



### ✅ 3. 시각적 탐색 기능



| 기능 | 설명 |

|------|------|

| 지도 기반 기업 분포 시각화 | 필터 조건에 맞는 기업들을 지도에 마커로 표시하여 **지역별 채용 집중도 확인** 가능 |

| 마커 정보창 | 지도상의 마커 클릭 시 해당 기업의 이름, 연봉 정보, 관련 공고 요약 표시 |

| 키워드 클라우드 | 기본요건/우대사항 키워드를 워드 클라우드로 시각화 |

| 연봉 그래프 | 필터링된 기업들의 연봉을 박스 플롯 또는 히스토그램 형태로 제공 |



---



## ➕ 추가/고도화 기능 제안



| 기능 | 설명 |

|------|------|

| 기업 문화 태그 분석 | 공고/기업 정보를 기반으로 "주 4일제", "유연근무제", "재택가능" 등의 문화 태그 제공 |

| 기업 비교 | 사용자가 선택한 기업들 간의 **요구 조건, 연봉, 지역, 공고 수** 비교 가능 |

| 공고 알림 설정 | 관심 산업군/직무에 맞는 신규 공고가 등록되면 알림 발송 (이메일 또는 웹 푸시) |

| 지원 이력 관리 | 사용자가 지원한 기업들을 기록하고 진행 상황을 관리할 수 있는 기능 |

| 마이페이지 기반 개인화 | 최근 열람 기업 기반으로 유사 기업 추천, 맞춤 통계 제공 등 |

## 🗂 예시 사용자 흐름 (User Journey)



```

[사용자] → 산업군: 핀테크 / 연봉: 4천~6천 / 지역: 서울로 필터

     ↓

→ 필터 조건에 해당하는 기업 85개 확인

     ↓

→ 상위 키워드: ‘SQL’, ‘AWS’, ‘Kafka’, ‘CS 전공’ 등이 요구됨

     ↓

→ 지도에서 강남·판교에 마커 밀집 → 특정 마커 클릭하여 기업 상세 확인

     ↓

→ 연봉 평균: 약 5,200만원, 주요 요건: 컴공 전공, 대규모 트래픽 경험

     ↓

→ 관심 기업 즐겨찾기 + 이메일 알림 설정

```

- 📊 연봉 분석 차트 (중앙 하단)  

- 🗺 지도 시각화 (우측)

## 📅 단계별 출시 로드맵 (MVP → 확장)



| 단계 | 목표 기능 |

|------|-----------|

| **1단계 (MVP)** | 공고/연봉 수집, 필터 기반 요건 통계, 지도 기반 기업 분포 |

| **2단계** | 연봉 분석 기능, 즐겨찾기/공고 알림, 마커 상세 보기 |

| **3단계** | 기업 비교, 개인화 통계, 지원 이력 관리, 커뮤니티 기능 |
